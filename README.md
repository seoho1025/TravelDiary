## 서비스 개요 
**"여행의 모든 순간을 서포트하는 AI 기반 맞춤형 여행 파트너"**

* **#Recall (복기) |** 사진, 동선 다이어그램, 타임라인을 결합해 여행의 기억을 입체적으로 상기 
* **#Instant (대안) |** 계획에 차질이 생겨도 AI 실시간 장소 추천으로 즉각적인 대안을 제시
* **#Story (공유) |** 동행자와 함께 쓰는 공동 일기로 서로 다른 시선의 추억을 하나로 완성

### #1. 기록의 번거로움과 심리적 부담
`작성 허들`이 높은 상황에서 여행 일기 작성의 지속성은 빠르게 감소하고 있습니다.

여행의 소중한 순간을 남기고 싶어 하는 의지는 높지만, 매번 직접 글을 쓰는 행위는 사용자에게 큰 귀찮음과 시간적 부담을 줍니다. 특히 사진 정리와 글 작성을 병행해야 하는 수필 형태의 일기는 심리적 진입장벽을 높여 기록을 중도에 포기하게 만드는 주된 원인이 됩니다.

### #2. 사진 촬영 중심의 파편화된 기록
`기억의 역설`이 발생하는 상황에서 사진만으로는 당시의 온전한 감정을 보존하기 어렵습니다.

설문 조사 결과, **65.9%**의 사용자가 사진 촬영만으로 기록을 대체하고 있지만, 그중 **96.7%**라는 압도적인 수치가 '사진만으로는 당시의 추억이 충분히 기억되지 않는다'고 답했습니다. 이는 사진이라는 시각 정보는 넘쳐나지만, 당시의 분위기와 감정이라는 핵심 정보는 유실되고 있음을 시사합니다.

### #3. 공유 기록 수단의 부재
`단편적 기록`에 머무는 상황에서 동행자와의 통합된 추억 형성이 저해되고 있습니다.

동일한 장소와 시간을 공유하더라도 사람마다 기억하고 느끼는 포인트는 각기 다릅니다. 하지만 현재는 이러한 서로 다른 시각과 감정을 하나의 완성된 기록으로 엮어 공유할 수 있는 적절한 서비스가 부족하여, 여행의 기록이 개인의 단편적인 경험에만 머물게 되는 한계가 있습니다.

## 서비스 방향성

> **"기록에서 계획으로, 계획에서 다시 추억으로 이어지는 선순환 여행 경험"**

### #1. 온전하고 입체적인 기록의 보존
`기록의 자동화`를 통해 사진 한 장에 담기지 않는 여행의 모든 순간을 온전히 복기합니다.

* **AI 기반 일기 생성 |** 사진의 위치·시간 및 감정을 분석하여 자동으로 일기 초안 작성
* **기억의 시각화 |** 여행 전체 경로를 다이어그램과 타임라인으로 형상화하여 세밀한 기억 환기 지원

### #2. 실시간 상황에 맞춘 유연한 대응
`흐름의 유지`를 목표로 예기치 못한 변수 상황에서도 여행의 즐거움이 끊기지 않도록 지원합니다.

* **지능형 경로 추천 |** 사용자의 현재 위치와 상황에 맞는 장소를 실시간으로 제안
* **즉각적 대안 제시 |** 예상치 못한 변수 발생 시 즉시 대안을 제공하여 여행의 흐름 보존

### #3. 함께 만드는 다각도의 추억 공유
`시각의 확장`을 통해 동행자와의 경험을 하나로 모아 더 풍부한 스토리를 구축합니다.

* **공동 일기 시스템 |** 동행자와 사진 및 느낀 점을 공유하여 하나의 통합된 스토리로 기록
* **기록의 인사이트화 |** 이전 여행 기록을 되살려 새로운 여행 계획의 유용한 참고 자료로 활용

## 💡 Solution
* **기억의 자동 연결 |** AI 기반 일기 생성 및 감정 분석 반영
* **여정의 시각화 |** 동선 다이어그램 및 타임라인 제공
* **공동 기록 시스템 |** 동행자와 함께 쓰는 실시간 공유 일기
* **지능형 경로 추천 |** 실시간 위치 기반 대안 장소 제안

### 프론트엔드 FRONTEND
| React Native, Expo, Zustand, TypeScript, Axios, React Navigation

#### **React Native & Expo**
* **크로스 플랫폼 개발 |** 단일 코드베이스 기반의 iOS 및 Android 통합 사용자 경험 환경 구축
* **기능 통합 최적화 |** Expo SDK를 활용한 카메라 및 위치 정보 등 모바일 하드웨어 기능의 신속한 연동

#### **Zustand**
* **전역 상태 관리 |** Flux 패턴 기반의 경량 상태 관리 라이브러리 도입을 통한 데이터 흐름 단순화
* **코드 유지보수성 향상 |** 프롭 드릴링 제거 및 컴포넌트 간 데이터 공유 로직 최적화

### 백엔드 BACKEND
| Spring Boot, Nginx, Redis, Supabase (PostgreSQL), Docker, Grafana Loki

#### **Spring**
* **RESTful API 구축 |** Java 17 기반 Spring Boot를 활용한 핵심 비즈니스 로직 개발
* **데이터 상호작용 최적화 |** Spring Data JPA 및 Querydsl 도입을 통한 효율적인 DB 쿼리 제어
* **협업 효율성 증대 |** Swagger를 활용한 명세 관리 및 프론트엔드 간 정밀한 데이터 통신 구현
* **DB 형상 관리 |** Flyway 적용을 통한 데이터베이스 스키마 변경 이력의 체계적 관리
